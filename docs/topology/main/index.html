位相

Chef : momoyuu

1. 概要

位相空間とは，集合に開集合系という付加構造が備わったものである．集合に開集合系が定まることで元同士の近さをはかることができるようになり（$f$,点列$x_n\to x$に対し$f$が連続であることと$f(x_n)$が$f(x)$に収束することが同値であったことを思い出されたい），それによって位相空間の間の連続写像が定義される．位相空間論は現代数学の基礎中の基礎であり，幾何，解析，代数のいずれにおいても姿を現す．また，位相空間論を学んでいくにつれて，いかに我々が先入観にとらわれて生活しているかが見え，日常生活においても視野を広げられるようになることだろう．（2231910）

2. 前提知識

集合論ゼミで行う集合論の知識については知っているものとして進める．特に，集合と写像の操作に慣れていることが好ましい．

3. 計画

  1. ユークリッド空間における開集合及び収束と連続写像
  2. 距離空間の定義とそこにおける開集合及び収束と連続写像
  3. 位相空間の定義と開集合及び近傍系(と収束)
  4. 位相空間の部分集合と連続写像
  5. 可算公理
  6. 誘導される位相と積空間(と箱位相、コンパクト開位相)
  7. 誘導される位相と直和空間・商空間(と商写像)、直和空間・商空間の普遍性と例
  8. 分離公理
  9. コンパクト性とチコノフの定理の紹介
  10. 連結成分、弧状連結成分
  11. 局所コンパクハウスドルフ空間
  12. 諸々の性質(分離公理と可算公理の関連性、距離空間との関係)
  13. 位相空間における収束(ネットとフィルター)
  14. パラコンパクト性
  15. 距離化可能性

4. 参考書

主に，

  • 梅原雅顕・一木俊助「これからの集合と位相」(裳華房)

を用いる．また必要に応じて，

  • 齋藤正彦「数学の基礎　集合・数。位相」(東京大学出版会)
  • 松坂和夫「集合・位相入門」(岩波書店)
  • 森田紀一「位相空間論」(岩波書店)

などを参照する．
