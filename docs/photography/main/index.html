写真

Chef : harry_arbrebleu

1. 概要

Jakobsonの6機能図式を背景に芸術を読解すれば，自己の内部言語の文法の探索を芸術の営みと定義できる． 芸術とは，言語の機能のうち，＜受け手＞が存在しないものである． つまり，自己の内部言語を試行錯誤を繰り返しながら探索する活動の過程で生まれるものが芸術作品である．

今日では，Instagramをはじめとするソーシャルメディアの発達と，iPhoneなどの高性能のカメラを備えたスマートフォンの普及により， インターネット上で様々な写真を目にするだろう， それは，以前は一部の芸術家の表現技法に過ぎなかった「写真」が市民権を獲得したことを意味するだけでなく， 写真は常に衆人の監視下に置かれ解釈され続けられるようになったことをも意味する．

さて，芸術としての「写真」を考えてみよう． 芸術作品は常に批評に晒されることは世の常であるが，作者が批評家に迎合して作品制作を行っていたとしたら， 果たして，それは芸術作品といえるのだろうか？ インターネット上に氾濫している数多の写真は，匿名であったり，実名であったりするものの，確かに存在する＜受け手＞を想定してはいないだろうか？ その写真は，芸術としての側面を失い，自然言語よりも不便で直観的に＜メッセージ＞を伝達する手段に堕ちてしまってはいないだろうか？

本ゼミの究極的な目標は，様々な理由により，大衆のコミュニケーションツールに没落しかけた「写真」を， 芸術の営みとして用いようとするレジスタンスの活動拠点となることである． 世界をシャッタ1つで分節化する「写真」は容易に試行錯誤ができるため，他の芸術表現を製作するのに比べて時間がかからない． 例えば「音楽」のように1曲を製作するのにかかる時間や，「小説」を一編書き上げるのにかかる時間に比べれば， 短時間で作品を作り上げることが出来る．この性質を活用して，毎回のゼミでは，1つ前のゼミで定めたお題に沿って製作した作品を参加者同士で批評を行ったあとに，毎回の担当者が，表現技法について解説する．

2. 前提知識

デジタルカメラを持っていることが望ましい． 機種は問わないが，スマートフォンよりもデジタルカメラを用いることを推奨する． なお，SFCメディアセンターでSFC生は3泊4日， SFC生以外は当日のみ，カメラとレンズを借りることが出来る．

3. 計画

春休み全体で7回程度の予定である． 各回の暫定的なテーマは以下の通り．

  1. 構図について(宿題: 紹介した構図のルールを用いた写真)
  2. 前回の宿題の批評，絞り・シャッタスピード・ISO感度について(宿題: 絞り・シャッタスピード・ISO感度について工夫した写真)
  3. 前回の宿題の批評，光について(宿題: 午後5時から翌午前7時までに撮影した写真)
  4. 前回の宿題の批評，色について(宿題: 色について現像で工夫した写真)
  5. 前回の宿題の批評，生き物の写真について(宿題: 人か動物の写真)
  6. 前回の宿題の批評，風景の写真について(宿題: 日常の風景の写真)
  7. 前回の宿題の批評，総括，(宿題: 非日常の写真)

各回の宿題は参加者の了承を得たうえで，ゼミのホームページに掲載する予定である．
